1 Create a Pier Model {#create_pier}
2 ======================================
3 Because piers are an integral part of PGSuper/PGSplice we don't have to look to far to find the pier model. The basic steps for defining the pier model in PGSuper/PGSplice are:
4 
5 1. Create a new PGSuper project.
6 2. Edit the bridge model so that it has at least two spans
7 3. Select *Edit > Abutment/Pier*. When prompted, select Pier 2. The Pier Details window will open.
8 4. Select the Layout tab
9 5. Set the Pier Model Type to Physical.
10 6. Enter the cap beam and column layout dimensions as well as the concrete material properties.
11 7. Select the Connections tab
12 8. Use the Boundary Condition option to select the super-to-substruture connectivity.
13 9. Enter the dimensions of the pier diaphragm or use the keyword "Compute" to have the dimensions computed based on the girder connection properties
14 
15 
16 > NOTE: Pier can be modeled as Physical or Idealized. Idealized pier do not have any dimensions, the represent simple support conditions only. Only the cross beams of Physical piers can be load rated
17 
18 
19 XBRate extends the user interface of PGSuper/PGSplice by adding the XBeam tab to the Pier Details window.
20 
21 
22 1. Select the XBeam tab
23 2. Enter the reinforcement type, condition factor, and enter the longitudinal reinforcement and stirrups.
24 3. Press [OK].
25 
26 
27 You probably noticed that the pier definition and XBeam input are very similar to XBRate in stand alone mode.
28 
29 
30 Select *View > Pier View* to view the pier model.
31 
32 
33 > TIP: The Pier View can only display piers that have been modeled as "physical" piers.
