Reinforced Concrete Cross Beam Load Rating
PGSuper/PGSplice Rating Options

XBRate adds a tab titled XBeam to the Load Rating Options window in PGSuper and PGSplice. Load rating options specific to cross beams are defined on this tab.

Item Description
When PGSuper is in Envelope Mode, use Select the analysis model to use for pier reactions when the Structural Analysis Method in PGSuper is set to "Envelope of Simple Span and Simple made Continuous Analysis".
ItemDescription
SimpleReactions are based on simple span analysis
ContinuousReactions are based on simple spans made continuous analysis
EnvelopeReactions are the greater of thos from simple span and a simple spans made continuous analysis
Compute permit rating factors using Select the method for computing permit rating factors
Maximum Live Load Step Size Enter the maximum live load step size
Maximum Number of Lanes Enter the maximum number of lanes to be considered in the analysis. Enter the keyword ALL to analyze for all lanes.

NOTE: See Load Rating Options, Analysis tab, for additional information on live load step size and number of lanes.
