1 Live Load Results {#appendix_a_live_load_results}
2 ============
3 The Live Load Results view provides a graphical representation of the live load analysis performed by XBRate.
4 
5 Item | Description
6 ------|------------
7 Response Selector | Select desired structural response; Moment or Shear
8 Live Load Type | Use the drop down list to select the general live load type such as Design, Legal, or Permit
9 Vehicle Type | Use the drop down list to select a specific vehicle for the selected general live load type
10 Live Load Configuration | Select a live load configuration
11 [<] and [>] | Press these buttons to move the live load left and right
12 Permit vehicle in lane | Select the lane in which the permit vehicle is located
13 
14 For Design and Legal live loads, as well as the Permit live load when the AASHTO MBE rating equation is used, three results graphs will be displayed:
15 1. Envelope for all of the vehicles associated with a Live Load Type
16 2. The response for a particular vehicle type
17 3. The response for a specific live load configuration
18 
19 When the WSDOT BDM rating equation is used for Permit live loads, two results graphs will be displayed:
20 1. The response for the permit vehicle in a single lane
21 2. The response for the legal vehicle in all other lanes
22 
23 
24 The live load configuration is schematically shown on the graph including the wheel line reactions and positions.
