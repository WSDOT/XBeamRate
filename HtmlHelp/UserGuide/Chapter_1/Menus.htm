Menus

All commands are available from the Main Menu. All the menus are shown in this section with a brief explanation and the associated icon from the toolbar.

File Menu

 

Command

Description

New

Creates a new BridgeLink Project

Open

Open an existing BridgeLink Project

 

Close

Close the current project

Save

Save the current project

 

Save As

Save the current project with a new name

Print

Print the current view

 

Print Setup

Setup the printer

 

Most Recent Projects (1,2,3,4,...)

Opens a recently used BridgeLink Project

 

Exit

Exit the BridgeLink Application

Edit Menu

 

Command

Description

Undo

Undo the most recent edit

Redo

Repeat the last edit that was undone

Pier

Edit the pier model

Project Menu

 

Command

Description

Update Now

Causes the output views to be updated if AutoCalc mode is disabled. You can also press the F5 key to update.

Units

Toggles the units mode

Load Rating Options

Edit the load rating options

 

Properties

Edit the project properties

Options Menu

 

Command

Description

 

Turn AutoCalc On/Off

Toggles AutoCalc mode

 

Reporting

Edit reporting options including the favorite reports list and creating custom reports

 

Hints

Manage the display of user interface hints

View Menu

 

Command

Description

 

Toolbar

Manage the application toolbars

 

Status Bar

Toggles the status bar on and off

 

Graphs

Displays a list of graph options. Select a graph option to display the graph.

 

Reports

Displays a list of reports. Select a report to create.

 

Pier View

Displays the Pier View

Window Menu

 

Command

Description

 

Cascade

Resizes and positions all windows in an overlapping pattern

 

Tile

Resizes and positions all windows in a non-overlapping pattern

 

Arrange Icons

Arranges all iconized windows in a grid at the bottom of the main window

 

Window List

Lists the open windows. Brings an open window to the top

Help Menu

 

Command

Description

 

Help Topics

Starts the on-line help system

 

About BridgeLink

Displays information about BridgeLink including copyright notices, version number, and application types

 

About XBRate

Displays information about XBRate including copyright notices and version number.
