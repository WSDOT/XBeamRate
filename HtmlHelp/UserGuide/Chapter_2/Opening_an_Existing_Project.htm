Opening an Existing Project

You can open existing BridgeLink projects using several techniques.

TIP: BridgeLink can only have one project open at a time. However, you can run multiple instances of BridgeLink, each with a different project.

Techniques for Opening BridgeLink Projects

Recent Projects List

The most common method of opening a BridgeLink project is to select it from the Recent Projects list located on the File menu.

TIP: Press the down arrow to the left of the Open button on the toolbar to quickly access the Recent Project List

Standard Open Command

The standard method of opening files in any Windows program is to use the File > Open command. This command displays the standard File Open window. Using this window, find a project file to open. Use the file type filter to display only files of the type you want to open.

Drag and Drop

If you locate your project file using Windows Exporter or My Computer, you can drag and drop it onto a running instance of BridgeLink to open the project.

Associated Files

If you locate your project file using Windows Exporter or My Computer, you can double click on it to start BridgeLink and open the project.
