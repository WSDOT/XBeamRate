NOTES:

 

If you define the pier diaphragm with W = 0 in PGSuper/PGSplice, XBeam will make the width equal to the width of the lower cross beam.

 

If you define the pier diaphragm with H = 0 in PGSuper/PGSplice, XBeam will make the height equal to Hgirder + "A" - Tslab.

 

A good technique is to use W and H = 0 for Physical Piers.
