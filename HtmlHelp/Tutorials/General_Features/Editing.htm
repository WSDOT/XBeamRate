Editing the Pier Model

The default pier model isn't exactly what we need. The geometry, materials, reinforcement, and loading don't represent our pier. The pier model will have to be edited.

Select Edit > Pier to edit the pier model. This brings up the Edit Pier window.

Note: The Edit Pier window discussed here is for a stand-alone analysis. The pier model information is presented slightly differently for an integrated analysis.

The Edit Pier window has 5 tabs that group together related information about the pier.

Edit Pier Window tab

Description

Superstructure

Used to define information about the bridge superstructure including connectivity, location, and geometry

Substructure

Used to define information about the substructure including the cross beam dimensions and column arrangement

Material and Reinforcement

Used to define the material properties and reinforcement

Permanent Loads

Used to define the load factors and superstructure reactions for permanent loads

Live Load

Used to define the live load factors and reactions.

Select each tab as its general contents are discussed below.

Superstructure Tab

The superstructure tab defines information about the bridge superstructure. The superstructure is defined as everything above the cross beam. This includes the dimensions of the pier diaphragm if one is used.

The superstructure definition begins with the general pier type. XBRate can model piers with Continuous, Integral, or Expansion type connections with the bridge superstructure.

The overall superstructure geometry at the pier is also defined on the superstructure tab. The geometry consists of the pier skew, deck elevation at the alignment, gross slab depth, curb lines, and pier diaphragm dimensions.

Tip: Pay attention to how dimensions are defined. Dimensions are measured normal to the alignment, in the plane of the pier and normal to the plane of the pier.

Substructure Tab

The substructure tab defines the cross beam and the column layout. Dimensions are in the plane of the pier.
